# Personal Finance Tracker

A simple and responsive Personal Finance Tracker built using HTML, CSS, JavaScript, and Chart.js.

## Features

* Add Income and Expense transactions
* Delete transactions
* Dashboard showing Income, Expense, and Balance
* Filter transactions by type
* Expense Category Pie Chart
* Monthly Income vs Expense Bar Chart
* Dark Mode
* Local Storage support
* Responsive Design

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Chart.js

## Project Structure

```text
Personal-Finance-Tracker/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.
4. Start tracking your finances.
